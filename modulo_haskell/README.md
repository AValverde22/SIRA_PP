## Ejecución del módulo Haskell

### Requisitos

Para ejecutar el módulo funcional se necesita tener instalado:
- GHC, compilador de Haskell.
- Visual Studio Code o cualquier editor de código.
- Extensión Haskell para Visual Studio Code (opcional).





Abrir una terminal desde la carpeta principal del repositorio y ejecutar: runghc modulo_haskell/Main.hs
(Importante: el programa debe ejecutarse desde la carpeta principal del repositorio, ya que utiliza la ruta `data/historial.csv`.)

Luego ingresar: estudiante1 o estudiante2
